Image-2-PDF

Compile your images into PDF files, simply by drag and drop.

Golang version Python version

Usage

  1. Prepare a directory with ONLY image files (supported format: JPG, PNG).
  2. Drag your directory and drop it to the compiled executable file.
  3. The PDF generated will have the same file name as the directory, with the order of the images the same as the order of file names alphabetically.

Go

Requirements

Go 1.11+

Build

Output folder: ./bin

Linux

make

# To clean output folder
make clean
